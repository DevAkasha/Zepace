# Zepace
## 프로젝트 소개 
**젭페이스에 오신것을 환영합니다.**
젭페이스는 유니티로 제작된 메타버스 플렛폼입니다.
우주테마의 공간에서 각종 상호작용을 할 수 있습니다.
젭페이스 내부에는 미니게임이 있습니다.
